# Configuration Examples

This directory contains example configurations for common use cases. Copy and adapt these to your needs.

## Quick Reference

| Example | Use Case | Complexity |
|---------|----------|------------|
| [minimal-github](minimal-github/) | Simplest GitHub Actions setup | Basic |
| [minimal-gitea](minimal-gitea/) | Simplest Gitea Actions setup | Basic |
| [production](production/) | Full production deployment | Advanced |
| [multi-org](multi-org/) | Multiple GitHub organizations | Advanced |

## Getting Started

1. Choose an example that matches your use case
2. Copy the example directory to your project
3. Update the configuration with your values
4. Follow the deployment steps in each example's README

## Example Structure

Each example includes:
- `flake.nix` - Complete Nix flake configuration
- `README.md` - Setup instructions specific to that example
- Additional files as needed for the use case
